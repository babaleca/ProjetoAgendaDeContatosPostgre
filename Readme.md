# Projeto Git

Este projeto ensina a usar o Git.

## Conteúdo

- Inicializar um repositório
- Fazer commits
- Criar branches
- Mesclar alterações
- Enviar para um repositório remoto

## Uso

1. `git init`
2. `git add .`
3. `git commit -m "Primeiro commit"`
4. `git branch -M main`
5. `git remote add origin <url>`
6. `git push -u origin main`
