# Agenda de Contatos — PostgreSQL

Aplicação desktop desenvolvida em Java com interface gráfica (Swing) e banco de dados PostgreSQL. Permite gerenciar contatos de forma simples, com operações de cadastro, edição, busca e exclusão.

## Tecnologias
- Java (Swing)
- PostgreSQL
- NetBeans IDE

## Funcionalidades
- Cadastrar novo contato (nome, telefone, email)
- Editar contato existente
- Buscar contato por nome
- Excluir contato

## Como rodar

### Pré-requisitos
- Java 11+
- PostgreSQL instalado e rodando

### Passos
1. Clone o repositório
```bash
   git clone git@github.com:babaleca/ProjetoAgendaDeContatosPostgre.git
```
2. Crie o banco de dados no PostgreSQL
3. Configure as credenciais de conexão em `src/conexao/Conexao.java`
4. Abra o projeto no NetBeans e execute
